Free Quick Reference

Embroidery
Quick Reference

Starting points for pull comp, density, needles, and stabilizers -- plus fast fixes for common problems. Covers 12 fabrics.

ⓘ  This is a simplified reference. The full Stitch System includes complete setup, digitizing logic, and more.

This quick reference gives you reliable starting points for the settings you use every job -- pull compensation, density, needle type, and stabilizer. Use it when you need a fast answer.

It does not cover underlay logic, sequencing rules, digitizing decisions, or advanced troubleshooting. Those are part of the full Stitch System, available on Etsy.

Quick Settings

Pull Comp & Density

Starting values by fabric category. Adjust in 0.05mm increments and test stitch each time.

Fabric Pull Comp Density Notes
Cotton / Twill 0.20mm 0.40–0.45mm Standard starting point
Denim 0.20–0.25mm 0.42–0.50mm Slow to 500–600 SPM
Nylon / Ripstop 0.15–0.20mm 0.40–0.45mm Adhesive stabilizer
T-Shirt / Jersey 0.35–0.40mm 0.42–0.48mm Cutaway always
Pique / Polo 0.35–0.40mm 0.40–0.48mm 0.35mm minimum
Beanie / Knit 0.35–0.45mm 0.40–0.48mm Firm cutaway
Fleece 0.25–0.35mm 0.50–0.60mm WS topping required
Terry / Towel 0.30–0.40mm 0.45–0.55mm WS topping always
Structured Cap 0.30–0.40mm+ 0.40–0.50mm Above flat baseline
Leather / Vinyl 0.10–0.15mm 0.50–0.60mm No hoop, no underlay

The default pull comp in most software is 0.17mm. This is too low for most fabrics outside of stable wovens. Leaving it at default on knits or caps will produce visible gaps every time.

Needle & Stabilizer

Needle by Fabric
Woven (cotton, denim) Sharp 75–90
Knit (jersey, polo) Ballpoint 75
Denim / heavy fabric Sharp 90–100
Cap / thick layers Sharp 80
Leather / vinyl Sharp or leather 75
Metallic thread Metallic eye 80
Stabilizer by Fabric
Cotton / nylon / denim Tearaway
Jersey / polo / knit Cutaway
Beanie / stretch Firm cutaway
Fleece / terry Cutaway + topping
Cap (structured) Tearaway (single)
Leather / vinyl Self-adhesive only

Never use tearaway on jersey, beanie, or any knit. Tearaway removes all structural support after the first wash. Cutaway only on all stretch and knit fabrics.

Troubleshooting

Common Problems

Tap a problem to see the likely cause and a quick fix.

Cause Pull comp too low. The default (0.17mm) is not enough for most fabrics.
Fix Raise pull comp by 0.05mm increments. Wovens: start at 0.20mm. Knits: 0.35–0.40mm. Test stitch each time.
Cause Pull comp too low specifically on the lettering object.
Fix Set pull comp on the text object independently. Do not change the whole design. Start at 0.20mm for wovens, 0.35–0.40mm for knits.
Cause Density too tight, wrong stabilizer, or no underlay on stretch fabric.
Fix Open density by 0.05mm. Switch to cutaway on knit. Float stretch fabrics -- do not hoop them under tension.
Cause No water-soluble topping on the pile surface.
Fix Place WS topping (Solvy) on the fabric surface before stitching. Add zigzag underlay to crush pile first. Not optional on terry.
Cause Speed too high, tension too tight, or the needle has a burr.
Fix Slow machine to 600 SPM. Reduce top tension in half-step increments. Run your fingernail over the needle eye -- if you feel a catch, replace it.
Cause Sharp needle piercing and deflecting knit loops instead of passing between them.
Fix Switch to a ballpoint 75/11 needle. Use cutaway stabilizer. Float the fabric rather than hooping under tension.
